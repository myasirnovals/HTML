Info Demo & Mapping Database

Pemetaan role dengan tabel pada database klinik.

  • ‚Ä¢ Pasien ‚Üí tabel pasien (field: id_pasien, nama_lengkap, alamat, tanggal_lahir, no_telepon, password)
  • ‚Ä¢ Dokter ‚Üí tabel dokter (field: id_dokter, nama_dokter, spesialis, no_telp) + password bisa tambah kolom baru di DB
  • ‚Ä¢ Staff ‚Üí tabel staff (field: id_staff, nama, password)
  • ‚Ä¢ Jadwal praktek ‚Üí tabel jadwal (relasi ke dokter)
  • ‚Ä¢ Rekam medis ‚Üí tabel rekam_medis (relasi ke dokter)
  • ‚Ä¢ Reservasi kunjungan ‚Üí tabel reservasi (relasi ke pasien, rekam_medis, staff, jadwal)
  • ‚Ä¢ Rujukan ‚Üí tabel rujukan (relasi ke rekam_medis)

Di demo ini, data login hanya disimpan di localStorage browser. Pada aplikasi asli, ganti logika di JavaScript dengan pemanggilan API / PHP yang terhubung MySQL.
